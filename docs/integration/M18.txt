M18: AskUserQuestion V2 Tool
============================

Status: Implemented
Date: 2026-04-02

Overview
--------
Enhanced AskUserTool with multi-question, options, preview, and
multi-select support. Validates and structures questions for the LLM.
The actual user interaction is handled by the interactive permission
flow, not this tool directly.

Architecture
------------
AskUserV2Tool is a READ-ONLY tool (PermissionLevel::Read) that:

1. Accepts a "questions" array (1-4 items)
2. Each question has: question string, options (2-4), optional header,
   preview, and multiSelect fields
3. Validates all inputs
4. Returns a structured JSON response with formatted questions
5. Returns "__WAITING_USER_INPUT__" marker for the agent loop

The tool is thread-safe (no mutable state).

Schema
------
Input:
  - questions: array (1-4 items)
    - question: string (required)
    - options: array (2-4 items, required)
      - label: string (required)
      - description: string (optional)
    - header: string (optional, max 12 chars)
    - multiSelect: boolean (optional, default false)
    - preview: string (optional)

Output:
  - answers: { Q1: { question, options, selectedOptions, ... } }
  - pause_agent: true
  - type: "ask_user_v2"
